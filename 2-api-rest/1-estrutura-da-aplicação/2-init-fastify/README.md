# Comandos

## Instalar o fastify

```bash
npm i fastify
```

## Instalar os types para o ts

```bash
npm i -D @types/node
```

## compilar para js

```bash
npx tsc src/server.ts
```

## Para não precisar ficar compilando e rodar direto o TS

Converte o ts para js de forma automática, apenas para desenvolvimento.

### Instalar o tsx

```bash
npm i -D tsx
```

### Rodando a api

```bash
npx tsx src/server.ts
```
